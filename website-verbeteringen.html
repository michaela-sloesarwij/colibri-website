🍪 1. Cookiebanner (EU-verplichting)

Plak deze code vlak vóór de sluit-tag </body> in alle HTML-pagina's (index, aanbod, over-mij, contact).

CSS + HTML + JS (alles in één blok)

<style>

<div class="cookie-bar" id="cookieBar">
  <p>🍪 Wij gebruiken functionele cookies om de website goed te laten werken. Wij plaatsen geen tracking- of advertentiecookies. <a href="/privacy.html">Meer info</a></p>
  <div class="cookie-btns">
    <button class="cookie-btn decline" onclick="cookieDecline()">Alleen noodzakelijk</button>
    <button class="cookie-btn accept" onclick="cookieAccept()">Akkoord ✦</button>
  </div>
</div>

<script>
(function(){
  if(!localStorage.getItem('csc_cookie_consent')){
    setTimeout(()=>document.getElementById('cookieBar').classList.add('show'),600);
  }
})();
function cookieAccept(){localStorage.setItem('csc_cookie_consent','accepted');document.getElementById('cookieBar').classList.remove('show');}
function cookieDecline(){localStorage.setItem('csc_cookie_consent','declined');document.getElementById('cookieBar').classList.remove('show');}
</script>
<!-- ═══ EINDE COOKIEBANNER ═══ -->
💡 Je hoeft geen externe cookie-service te betalen — jouw website gebruikt geen tracking cookies, dus deze simpele melding is voldoende voor de EU wet.

🌍 3. Taalswitch repareren (NL/EN)

Het probleem: de EN-link verwijst naar pagina's die nog niet bestaan. Twee opties:

Optie A — Tijdelijk: taalswitch verbergen tot EN klaar is

<!-- Vervang je huidige taalswitch door dit -->
<div class="nav-lang">
  <a href="/index.html" class="lang-active">NL</a>
  <span style="opacity:.4" title="Engels komt binnenkort">| EN</span>
</div>

Optie B — Correcte links naar /en/ map (aanbevolen)

Maak een map /en/ in je GitHub repo met: index.html, offer.html, about.html, contact.html

<!-- Nav op NL-pagina's -->
<div class="nav-lang">
  <a href="/index.html" class="lang-active">NL</a> |
  <a href="/en/index.html">EN</a>
</div>

<!-- Nav op EN-pagina's (/en/ map) -->
<div class="nav-lang">
  <a href="/index.html">NL</a> |
  <a href="/en/index.html" class="lang-active">EN</a>
</div>
⚠️ Gebruik Optie A zolang de EN-pagina's nog niet klaar zijn. Gebroken links schaden je SEO.

📋 5. Stappenplan — volgorde van aanpakken

1
Open je GitHub repo en ga naar index.html
2
Voeg de SEO meta-tags toe aan <head> van alle pagina's
3
Plak de cookiebanner code vóór </body> op alle pagina's
4
Vervang contact.html met het nieuwe bestand (heeft formulier + kaart)
5
Formspree.io — maak gratis account, maak een form aan, kopieer het ID, plak het in contact.html bij YOUR_FORMSPREE_ID
6
Repareer de taalswitch (Optie A is snelst)
7
Voeg de opleiding-sectie toe aan over-mij.html
8
Meld je website aan bij Google Search Console voor betere vindbaarheid
